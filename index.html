MS THESIS · 2026–2027

Multi-Modal AI Content Detection Framework

A unified detection system spanning three modalities — written text, source code, and images — that analyzes forensic signals to distinguish AI-generated content from human-authored work. Features a false positive reduction layer protecting students from erroneous AI accusations, and an assignment context module that detects mismatches between what a teacher assigned and what a student submitted. Deployed as a free public web tool with no login required.

SENIOR DESIGN · 2024–2025

Emotionally-Aware AAC System for Nonverbal Neurodivergent Children

Real-time pipeline detecting emotion from facial expressions using a custom CNN, then modulating text-to-speech prosody so synthesized speech sounds emotionally expressive. Built for nonverbal children who cannot speak. A core challenge was that standard emotion recognition datasets are overwhelmingly neurotypical adults — we made deliberate architectural decisions including collapsing Fear and Disgust into Neutral to reduce harmful misclassification.
